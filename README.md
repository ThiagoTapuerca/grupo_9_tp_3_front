# Raiz del proyecto

index.html es el punto de entrada a la pagina de inicio

.gitignore es el archivo que indica a Git los archivos y/o carpetas no deben subir al repo

# Carpeta /pages

contacto.html: datos de contacto.

equipo.html: Información sobre los integrantes del grupo.

faq.html: Sección de preguntas frecuentes.

pedido.html: Interfaz para la realización de los pedidos.

stock.html: Visualización del inventario/stock disponible.

# CSS

style.css: Estilos generales aplicados a todo el sitio y al index.html.

styleContacto.css, styleEquipo.css, stylePedidos.css, styleStock.css son archivos específicos que contienen el diseño para cada página de la carpeta /pages.
